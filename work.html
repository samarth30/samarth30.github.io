Vibin

soon avaliable on playstore

Vibin is basically a mixture of youtube and spotify where you can search your videos on youtube or songs from spotify and save into your playlist.In this app you can create your own playlist, save songs and videos in one playlist, set playlist private or public and can search your and others playslist two.The private feature of playlist can safeguard your personal songs and videos that you have added.You can also add as a collaborator to a private or public playslist and All this is going to be stored in a rest apis database made eith django it also includes the login signup page and a logout too.This app used the youtube and spotify api to access the content.

Technologies Used

Android java youtube-api spotify-api rest-apis UI/UX Database Django
